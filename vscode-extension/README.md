# Codescope VS Code Extension

Code intelligence powered by knowledge graphs.

## Features
- **CodeLens**: See caller/callee counts above functions
- **Commands**: Index, search, and stats from command palette
- **MCP Integration**: Communicates with codescope-mcp via stdio

## Requirements
- codescope-mcp in PATH (`npm i -g codescope` or install from GitHub releases)

## Development
npm install
npm run compile
# Press F5 to launch extension development host
